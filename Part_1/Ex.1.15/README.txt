This is a dockerfile for the repository for github repository https://github.com/vornsami/kuvalauta

The dockerfile makes the application compatible with docker, so no changes are necessary.

Place the dockerfile in the project root.

The Docker image can be built with the command "docker build -t kuvalauta ." and run with the command "docker run -d -ti -p 5000:5000 kuvalauta".

The application is deployed to localhost:5000.
